OSI七层模型结构, TCP/IP四层模型学习笔记

发表于 2018-06-12 | 分类于 网络

OSI七层和TCP/IP四层的关系

  1. TCP/IP先有协议再提出的模型,是参照OSI模型
  2. OSI是理论下的模型,而TCP/IP已被广泛使用
  3. TCP transmission control protocol 传输控制协议
  4. UDP user data protocol 用户数据报协议

OSI七层模型

OSI 功能 协议
应用层 面向计算机用户,提供应用程序和网络之间的接口 HTTP, HTTPS, FTP, TELNET, SSH, SMTP, POP3等
表示层 处理来自应用层的的命令和数据,结果传给会话层 N/A
会话层 不同机器上的用户之间建立会话 N/A
传输层 为会话层和传输层提供传输服务 TCP, UDP
网络层 IP等
数据链路层 数据链路层的数据传输单元成为帧, 就是将网络层转发的数据封装为帧, 传输给物理层, 以及分析物理层传输过来的数据帧 SLIP, CSLIP, PPP, ARP, RARP, MTU等
数据层 ISO2110等

TCP模型

OSI TCP/IP 协议
应用层/表示层/会话层 应用层 HTTP, FTP, TFTP, SMIP, SNMP, DNS
传输层 传输层 TCP, UDP
网络层 网络层 ICMP, IGMP, IP, ARP, RARP
数据链路层/物理层 网络接口层 由底层网络定义协议

交互实例

发送端每通过一层则增加首部,接收端每通过一层则删除首部
|交互|发送端|接收端|
|:-:|:-|:-|
|应用层|HTTP数据|HTTP数据 |
|传输层| TCP首部(HTTP数据) | TCP首部(HTTP数据) |
|网络层|IP首部(TCP首部(HTTP数据) | IP首部(TCP首部(HTTP数据) |
|链路层| 以太网首部(IP首部(TCP首部(HTTP数据))) | 以太网首部(IP首部(TCP首部(HTTP数据))) |
