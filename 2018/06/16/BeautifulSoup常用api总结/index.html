BeautifulSoup常用api

发表于 2018-06-16 | 分类于 python

解析器

Beautiful Soup支持python标准库中的HTML解析器,还支持一些第三方的解析器,常见的解析器有
|解析器|使用方法|
|:-:|:-:|
|python标准库|BeautifulSoup(markup,’html.parser’)|
|lxml HTML解析器|BeautifulSoup(markup,’lxml’)|
|lxml XML解析器|BeautifulSoup(markup,’lxml’, ‘xml’)|
|html5lib|BeautifulSoup(markup,’html5lib’)|

基本使用

  1. soup.p获取第一个p标签
  2. soup.p.attrs['class']或soup.p['class']获取第一个p标签的name属性的值
  3. soup.p.string或soup.p.text获取标签的文本属性
  4. soup.p.children获取第一个p标签的子元素(数组形式)
  5. soup.a.parent获取第一个a标签的父元素
  6. soup.a.next_siblings获取后面的所有兄弟元素,soup.a.next_sibling获取下一个兄弟元素,soup.a.previous_siblings相同
  7. soup.title获取title的内容,soup.title.name获取标签名称

    标准选择器find

  8. find_all返回列表(数组)
    1. soup.find_all('ul')获取所有ul标签以及每个ul标签的内部标签
    2. soup.find_all(attrs={'name': 'xxx'})获取所有属性name为xxx的标签
    3. soup.find_all(text='xxx')获取文本为xxx的标签
  9. find(name, attrs, recursive, text, **kwargs)返回第一个元素

CSS选择器

  1. soup.select('ul')返回所有ul标签
  2. soup.select('li')[0].get_text()获取文本内容
  3. soup.select('ul')[0].attrs['id']获取第一个ul的id属性值
