JavaScript学习笔记-数据类型

发表于 2018-06-17 | 分类于 JavaScript

JavaScript数据类型

浅析JavaScript语言的类型

数字number

  1. js中的Number类型使用的是双精度浮点型,也就是其他语言的double类型
  2. JS number相加精度丢失
  3. var a = 'a' - 1;NaN
  4. NaN于自己不相等
  5. isNaN()返回布尔值,内部会先使用number()转化

字符串string

字符串方法(不会修改原字符串)

  1. str.length只可读
  2. str[i]不兼容IE6-7(str.charAt(i))
  3. str.indexOf(value, index)从index位(默认为0)开始检索,返回第一个value的下标或-1
  4. str.lastIndexOf(value, index)从最后一位开始检索,返回第一个value的下标或-1
  5. str.charAt(index)返回执行序号的字符串
  6. str.charCodeAt(index)返回执行序号下字符的ASCII码
  7. String.fromCharCode(80)返回ASCII为80的字符
  8. str.substring( 起始序号-包含 , 结束序号-不包含 )返回截取的字符串
  9. str.slice( 起始序号-包含 , 结束序号-不包含 )返回截取的字符串,负数代表倒数的index
  10. str.substr( 起始序号-包含 , 截取长度 )返回截取的字符串
  11. str.trim去除首尾空格
  12. str.toUpperCase() / str.toLowerCase()
    以下方法可以使用正则
  13. str.match('c')返回c或null
  14. str.search('c')返回str中第一个c的下标或-1
  15. str.replace('c','d')只替换第一个(如果使用正则的全局匹配就可替换所有),返回替换后的str,
  16. str.split('c')返回用c将str分割成的数组

布尔值boolean

对象Object

  1. 可以进行点操作的就是对象
  2. 有js定义的属性,也有自定义对象的属性
  3. 查看对象下的属性
    1. for in

      节点对象,json,array,window,null…

JSON

  1. 声明
    1. 直接量
    2. new形式
  2. json中的this
    1. var person = {'name': this, 'a': function(){this}}
    2. name对应的this指window
    3. 函数中的this指这个json
  3. 删除属性
    1. delete person.name
  4. in判断对象是否具有某种属性
    1. 'name' in person返回布尔值
  5. 遍历json
    1. for (var attr in json){}
  6. JSON.parse(str) / JSON.stringify(obj)

数组

数组的方法
  1. 数组.length 可读可写
  2. 数组[序号] 可读可写
  3. arr.unshift( 新数据1 , 新数据2 )
    功能: 在原数组开头添加新数据
    返回: 新数组的长度
    是否改变原数组: 是
  4. arr.push( 新数据1 , 新数据2 )
    功能: 在原数组最后添加新数据
    返回: 新数组的长度
    是否改变原数组: 是
  5. arr.shift()
    功能: 移除数组第一个成员
    返回: 移除的成员
    是否改变原数组: 是
  6. arr.pop()
    功能: 移除数组最后一个成员
    返回: 移除的成员
    是否改变原数组: 是
  7. arr.splice( a , b , c1 , c2 ... )
    a: 开始的序号
    b: 删除的个数
    c: 填充的成员
    返回: 只返回删除的对象
    是否改变原数组: 是
    只填一个参数时: 表示从下标为0处开始删除x个
  8. arr.slice( a , b )
    1. 与字符串splice相同
    2. 返回: 切割的数组
    3. 是否改变原数组: 不会
  9. arr.sort(function(x,y){return x-y})
    功能: 对数字数组排序
    return x-y : 原数组升序
return 小于0的数 : 原数组不变
return y-x : 原数组倒序
return 大于0的数 : 原数组反转
    返回: 新数组
    是否改变原数组: 是
  10. arr.reverse()
    功能: 反转数组
    返回: 新数组
    是否改变原数组: 是
  11. 数组1.concat( 数组2 , 数组3 )
    功能: 拼接数组
    返回: 新数组
    是否改变原数组: 否
  12. arr.join( 字符串 )
    功能: 将数组成员用参数字符串拼接成字符串
    返回: 字符串
    是否改变原数组: 否
  13. Array.isArray( 参数 )
    功能: 判断参数是否为数组
    返回: 布尔值

  14. arr.forEach(function(value,index,arr){})
    功能: 遍历数组
    是否改变原数组: 是

  15. arr.filter(function(value,index,arr){})
    return 条件
    功能: 筛选数组
    返回: 新数组
    内部函数返回布尔值
    是否改变原数组: 否

  16. arr.map(function(value,index,arr){})
    return 条件
    功能: 判断成员是否满足条件
    返回: 布尔值组成的新数组
    return 运算 : 返回运算后的新数组
    功能: 成员进行计算
    返回: 运算后的新数组
    是否改变原数组: 否
    return 值: 将数组每一项替换成这个值并返回

  17. arr.every(function(value,index,arr){})
    return 条件
    功能: 判断成员是否满足条件
    返回: 所有成员满足条件则返回true,否则返回false
    是否改变原数组: 否
  18. arr.some(function(value,index,arr){})
    return 条件
    功能: 判断成员是否满足条件
    返回: 只要有一个成员满足条件则返回true,否则返回false
    是否改变原数组: 否
